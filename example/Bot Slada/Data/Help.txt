Информация -- Информация о боте
Помощь -- Список всех основных команд
Помощь 2 -- Список всех развлекательных команд

Установить -- Регистрирует беседу в базе данных, позволяя использовать остальные админ-функции

АДМИНИСТРИРОВАНИЕ

Кик <ID пользователя, или его имя> -- Удаляет пользователя из беседы
Бан <ID пользователя, или его имя> [Причина] -- Удаляет пользователя из беседы. При попытке зайти в беседу, этого пользователя будет кикать.
Разбан <ID пользователя, или его имя> -- Снимает бан с пользователя
Предупреждение <ID пользователя, или его имя> -- Дает предупреждение пользователю. Если будет 3 предупреждения, пользователя удалит из беседы
СнятьПред <ID пользователя> -- Снимает одно предупреждение
Мут <ID пользователя, или его имя> -- За каждое сообщение пользователя, ему будет даваться предупреждение
Размут <ID пользователя, или его имя> -- Убирет мут у пользователя
АнтиВыход -- Включает/Отключает функцию антивыхода. При выходе из беседы, пользователя мгновенно кикает, чтобы он не мог зайти обратно.
Название <Название беседы> -- Изменить название беседы

УПРАВЛЕНИЕ БОТОМ

Админ <Уровень> <ID пользователя, или его имя> -- Изменяет ранг пользователя на нужный
Права <Команда> <Ранг> -- Устанавливает необходимый ранг для данной команды
РангИмя <Ранг> <Имя> -- Изменяет имя ранга
РангДобавить <Количество рангов> [Их имена, через запятую] -- Добавляет новые ранги
РангУдалить <Количество рангов> -- Удаляет ранги
РазвлКоманды -- Включает/Отключает развлекательные команды
СписокРангов -- Показать список всех рангов
СписокКоманд -- Показать список команд
